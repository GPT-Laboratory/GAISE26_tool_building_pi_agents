# Workshop Task: Analyze sample.csv

The dataset is at `data/sample.csv`. Work through these questions:

1. **Explore** — How many rows and columns? What are the data types?
2. **Summarize** — What are the min, max, and mean of the `value` column?
3. **Group** — What is the average `value` per `category`?
4. **Filter** — Which rows have a `value` above the overall mean?
5. **Visualize** (bonus) — Can you describe what a bar chart of average value by category would show?

Ask the agent to help you with each step. Try to understand the code it writes, not just run it.
